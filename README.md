# QR Code Component Extension - QR Code Generator
This is the React extension of the [QR Code Component](https://www.frontendmentor.io/solutions/qr-code-component-xixkp7hzyW) challenge from [Frontend Mentor](https://www.frontendmentor.io/challenges/qr-code-component-iux_sIO_H). This project transforms the original static design into a fully interactive QR code generator with customization and management features.

## Features
### Core Functionality
- **Dynamic QR Generation:** Automatically generates QR codes as you type.
- **Visual Customization:** Adjust foreground/background colors and size.
- **File Management:** Save, load, and organize multiple QR codes.
- **Export Options:** Download QR codes with custom filenames.
- **Local Persistence:** All saved QR codes are stored in your browser.

### Default Settings
- **Value:** `https://www.frontendmentor.io/`
- **Colors:** White (`#ffffff`) foreground on blue (`#2c7dfa`) background.
- **Size:** 55% of the container's width.
- **Filename Format:** `qrcode-YYYYMMDDHHMMSS` (ISO timestamp)
- **Storage:** Browser local storage for persistence
